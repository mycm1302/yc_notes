## 🎯 주요 방법

### 관절공간 제어
- **장점**: 직접적, 안정적
- **단점**: 작업 좌표 제어 어려움
- **적용**: 점대점 운동

### 작업공간 제어
- **장점**: 직관적, 작업 친화적
- **단점**: 복잡, 특이점 문제
- **적용**: 연속 경로 추종

### 계산토크 제어
- **원리**: 동역학 모델로 비선형성 제거
- **형태**: τ = M(θ)[θ̈_d + Kv ė + Kp e] + C(θ,θ̇)θ̇ + G(θ)
- **특징**: 선형 시스템으로 변환
