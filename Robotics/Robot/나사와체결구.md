## 🔩 나사의 기하학

### 나사산 형태
- **미터 나사**: 60° 삼각 나사산
- **인치 나사**: UNC, UNF 규격
- **사각 나사**: 동력 전달용
- **사다리꼴 나사**: Acme 나사

### 나사 용어
- **피치 (p)**: 인접 나사산 간격
- **리드 (l)**: 1회전당 전진거리
- **나사산 각 (α)**: 60° (일반 나사)

---

## ⚖️ 토크와 축력 관계

```
T = F(dm/2)[(l + πμdm)/(πdm - μl)]

T: 조임 토크
F: 축력 (인장력)
dm: 나사 평균 직경
l: 리드
μ: 마찰계수
```

---

## 💪 볼트 연결 강도

### 예하중과 외하중
- **예하중 (Fi)**: 초기 조임력
- **외하중 (P)**: 작용 외력
- **볼트 축력**: Fb = Fi + P/n

### 안전계수
```
n = Sp·At·Se/(Fb·Sp + Se·Fi)

Sp: 증명하중
At: 인장단면적  
Se: 피로한도
```
