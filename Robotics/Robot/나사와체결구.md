## 🔩 나사의 기본 개념

### 나사의 정의와 기능
**나사 (Thread):**
- 원통면 또는 원뿔면에 나선형으로 만든 홈
- 직선운동을 회전운동으로 변환
- 체결, 조정, 전동 기능

**나사의 분류:**
```
용도별:
- 체결용 나사: 볼트, 너트, 목나사
- 전동용 나사: 리드 스크류, 볼 스크류
- 조정용 나사: 마이크로미터, 잭

방향별:
- 우나사: 시계방향 회전으로 전진
- 좌나사: 반시계방향 회전으로 전진
```

### 나사의 기하학
```
피치 (p): 인접한 나사산 사이의 축방향 거리
리드 (l): 1회전당 축방향 이동거리
- 1조 나사: l = p
- n조 나사: l = n × p

나사산 각 (α): 나사산의 포함각
- 미터 나사: 60°
- 유니파이 나사: 60°
- 아크미 나사: 29°
- 사각 나사: 90°
```

## 📐 표준 나사 규격

### 미터 나사 (ISO Metric)
```
표기법: M8×1.25
M: 미터 나사 기호
8: 공칭 직경 (mm)
1.25: 피치 (mm)

피치 구분:
- 거친 피치: 표준 피치 (피치 생략 가능)
- 가는 피치: 명시 필요

예: M8 (피치 1.25), M8×1.0 (가는 피치)
```

### 유니파이 나사 (Unified Thread)
```
표기법: 1/4-20 UNC
1/4: 공칭 직경 (inch)
20: 인치당 나사산 수 (TPI: Threads Per Inch)
UNC: 거친 피치 (Unified National Coarse)
UNF: 가는 피치 (Unified National Fine)

피치 계산: p = 1/TPI (inch)
```

### 주요 나사 치수
```
미터 거친 나사:
M3×0.5, M4×0.7, M5×0.8, M6×1.0
M8×1.25, M10×1.5, M12×1.75, M16×2.0

인치 거친 나사 (UNC):
#4-40, #6-32, #8-32, #10-24, #12-24
1/4-20, 5/16-18, 3/8-16, 1/2-13
```

## 🔧 나사의 역학

### 나사의 마찰
```
토크-축력 관계:
T = F × [dm/2 × (l + πμdm)/(πdm - μl)] + Tc

T: 조임 토크 (N·m)
F: 축력 (N)
dm: 나사 평균 직경 (m)
l: 리드 (m)
μ: 나사면 마찰계수
Tc: 좌면 마찰토크 (N·m)
```

### 간단한 근사식
```
표준 체결구:
T ≈ 0.2 × F × d

d: 공칭 직경 (m)
이 식은 일반적인 조임 토크 추정에 사용
실제 값은 윤활, 표면조도에 따라 변동
```

### 효율과 자동잠금
```
나사 효율: η = tan(α)/(tan(α + φ))

α: 나사각 (리드각)
φ: 마찰각 (φ = tan⁻¹μ)

자동잠금 조건: α < φ
- 사각나사: 높은 효율, 자동잠금 어려움
- V나사: 낮은 효율, 자동잠금 확실
```

### 리드각 계산
```
리드각: α = tan⁻¹(l/(πdm))

대부분의 체결용 나사: α = 2~5°
전동용 나사: α = 10~30°
급속 나사: α = 45° 내외
```

## ⚖️ 볼트 연결의 역학

### 볼트 연결의 특성
```
예하중 (Fi): 초기 조임력
- 볼트를 인장시켜 접합재를 압축
- 외하중이 없어도 볼트에 작용하는 하중

외하중 (P): 접합부에 작용하는 분리력
- 볼트 축력 증가 + 접합재 압축력 감소
```

### 강성과 하중 분담
```
볼트 강성: kb = Ab×Eb/lb
접합재 강성: km = Am×Em/lm

강성비: C = kb/(kb + km)

외하중 작용 시:
볼트 하중 증가: ΔFb = C×P
접합재 하중 감소: ΔFm = (1-C)×P

총 볼트 하중: Fb = Fi + C×P
```

### 볼트 단면적
```
인장단면적 (At): 인장강도 계산용
- 미터나사: At ≈ π/4 × (d - 0.9382p)²
- 인치나사: At ≈ π/4 × (d - 0.9743/n)²

나사부 단면적 (As): 전단강도 계산용
전체 단면적 (A): 압축강도 계산용

d: 공칭직경, p: 피치, n: TPI
```

## 💪 볼트의 강도 설계

### 정적 강도
```
인장강도: σt = Fb/At ≤ Sp/n

Fb: 볼트 축력
At: 인장단면적  
Sp: 증명하중 (Proof Load)
n: 안전계수 (2.5~4.0)

증명하중: 0.2% 영구변형이 발생하는 하중
```

### 볼트 재료와 강도 등급
```
미터 볼트 등급 (예: 8.8):
- 첫 번째 숫자×100: 인장강도 (MPa)
- 두 번째 숫자×10: 항복비 (%)
- 8.8급: Su = 800 MPa, Sy = 640 MPa

인치 볼트 등급:
- Grade 5: Su = 120 ksi, Sy = 92 ksi
- Grade 8: Su = 150 ksi, Sy = 130 ksi
```

### 피로 강도
```
변동 하중에 대한 피로 설계:
볼트 피로한도: Se ≈ 18 ksi (미가공 나사부)

굿맨 선도: σa/Se + σm/Su ≤ 1/n

σa = C×Pa/At (변동하중 진폭)
σm = (Fi + C×Pm)/At (평균응력)

Pa: 외하중 진폭
Pm: 외하중 평균값
```

## 🔩 너트와 체결 방법

### 너트의 종류
**6각 너트:**
- 일반형: 표준 두께
- 얇은형: 공간 제약시
- 후판형: 고하중용

**플랜지 너트:**
- 좌면이 넓어 압력 분산
- 와셔 기능 일체형

**로크 너트:**
- 자동잠금 기능
- 진동에 의한 풀림 방지

### 나사 풀림 방지
```
원인:
- 진동에 의한 미끄러짐
- 온도 변화에 의한 열팽창
- 소성변형에 의한 하중 손실

대책:
1. 마찰 증가: 스프링 와셔, 톱니 와셔
2. 기계적 구속: 나일론 인서트, 더블 너트
3. 화학적 고정: 나사 고정제 (록타이트)
4. 영구 변형: 캐슬 너트 + 코터핀
```

### 토크-텐션 관계 관리
```
토크 조임법:
- 가장 일반적
- 정확도: ±25% (마찰계수 변동)

각도 조임법 (Turn-of-Nut):
1. 손조임으로 접촉
2. 정해진 각도만큼 추가 회전
3. 정확도: ±10%

인장 조임법:
- 축력 직접 측정
- 가장 정확하지만 복잡
```

## 🔧 특수 체결구

### 캡 스크류 (Cap Screw)
```
용도: 정밀 기계 조립
특징: 원통형 헤드, 6각 소켓
재질: 고급강, 스테인리스강
강도: 12.9급까지 가능

장점: 미려한 외관, 높은 강도
단점: 비싸고 특수 공구 필요
```

### 셋 스크류 (Set Screw)
```
용도: 축에 기어, 풀리 고정
끝단 형상:
- 컵 포인트: 일반용
- 콘 포인트: 정밀 위치 결정
- 플랫 포인트: 경질 축용

토크 전달 능력:
T = μ×F×r (μ: 마찰계수, F: 압입력, r: 반지름)
```

### 엄지 나사 (Thumb Screw)
```
용도: 수동 조작이 필요한 곳
특징: 손으로 돌릴 수 있는 헤드
재질: 황동, 스테인리스강
크기: M3~M12 일반적
```

### 아이 볼트 (Eye Bolt)
```
용도: 인양, 견인용
설계 고려사항:
- 축방향 하중만 허용
- 측방향 하중시 강도 급감
- 안전계수 6~8 적용

사용법:
- 완전 체결 후 사용
- 정기적 점검 필요
```

## 📏 나사 가공과 측정

### 나사 가공법
**절삭 가공:**
- 선반 가공: 외나사
- 태핑: 내나사 (탭 사용)
- 다이스: 외나사 (다이 사용)
- 밀링: 대량 생산

**성형 가공:**
- 압연: 고강도, 대량 생산
- 단조: 특수 형상
- 주조: 일체형 나사

### 나사 정밀도
```
JIS 나사 등급:
1급: 정밀급 (게이지 나사)
2급: 중급 (일반 기계)
3급: 거친급 (일반 체결)

허용차:
- 1급: ±0.05mm
- 2급: ±0.1mm  
- 3급: ±0.2mm
```

### 나사 측정법
```
피치 측정: 나사 피치 게이지
유효직경: 3침법, 나사 마이크로미터
나사각: 공구현미경, 투영기
표면조도: 조도계

검사 기준:
- 피치 오차: ±0.02mm 이내
- 나사각 오차: ±30분 이내
- 표면조도: Ra 3.2μm 이하
```

## 🔨 특수 나사 시스템

### 사각 나사와 아크미 나사
```
사각 나사:
- 나사각: 90°
- 효율: 최대 95%
- 용도: 정밀 이송장치
- 단점: 가공 어려움, 마모시 백래시

아크미 나사 (Acme):
- 나사각: 29°
- 효율: 70~85%
- 용도: 잭, 바이스, C-클램프
- 장점: 가공 용이, 강도 우수
```

### 대각나사 (Buttress Thread)
```
특징:
- 비대칭 나사각
- 한 방향 하중에 특화
- 45°/7° 조합

용도:
- 유압 실린더
- 포신 체결
- 대형 구조물
```

### 파이프 나사
```
NPT (National Pipe Thread):
- 테이퍼 나사: 1/16 테이퍼
- 밀봉성 우수
- 배관 연결용

BSPT (British Standard Pipe Thread):
- 유럽 표준
- 55° 나사각
```

## 🔍 나사 연결 설계 실무

### 다중 볼트 연결
```
하중 분담:
- 이상적: 균등 분담
- 실제: 강성이 큰 볼트에 집중

개선 방법:
1. 볼트 강성 균등화
2. 접합재 강성 증가
3. 적절한 배치 (대칭성)
4. 예하중 관리
```

### 가스켓이 있는 연결
```
가스켓 강성: 매우 낮음
특성:
- 볼트가 대부분의 외하중 부담
- 누설 방지가 목적
- 과도한 압축 피해야 함

설계 요점:
- 적절한 가스켓 압력 유지
- 온도 변화 고려
- 크리프와 릴랙세이션 고려
```

### 고온 연결부 설계
```
문제점:
- 열팽창 차이
- 크리프와 릴랙세이션
- 재료 강도 저하

대책:
- 열팽창 계수 매칭
- 내열성 재료 사용
- 스프링 와셔로 하중 유지
- 정기적 재조임
```

## 💰 경제성과 표준화

### 표준 체결구의 장점
```
경제성:
- 대량 생산으로 단가 절감
- 재고 관리 용이
- 호환성으로 유지보수 편리

설계 효율:
- 설계 시간 단축
- 검증된 신뢰성
- 공급업체 다양성
```

### 특수 체결구 고려사항
```
필요성 검토:
1. 표준품으로 불가능한가?
2. 성능 향상이 확실한가?
3. 비용 증가가 정당한가?

개발 비용:
- 설계비, 금형비
- 시험비, 인증비
- 재고 비용 증가
```

### 라이프사이클 비용
```
LCC = 구매비 + 설치비 + 유지비 + 폐기비

고급 체결구:
- 초기 비용 높음
- 수명 길고 신뢰성 우수
- 유지비 절감

일반 체결구:
- 초기 비용 낮음
- 정기 교체 필요
- 총 비용은 고급품과 유사할 수 있음
```
