# 선형대수

> 상위: [[선형대수및기하학]]

벡터공간과 선형변환의 기본 이론을 다루는 수학 분야입니다. 로봇공학에서 좌표변환, 운동학, 동역학 해석의 핵심 도구로 활용됩니다.

## 📐 기본 개념

### 정의
선형대수학(Linear algebra)은 벡터 공간, 벡터, 선형 변환, 행렬, 연립 선형 방정식 등을 연구하는 대수학의 한 분야입니다. 현대 선형대수학은 그중에서도 벡터 공간이 주 연구 대상입니다.

**핵심 원리:**
- 덧셈과 상수곱 구조를 갖고 있는 벡터공간
- 선형성(Linearity): f(ax + by) = af(x) + bf(y)
- 연립방정식을 행렬로 표현하여 체계적으로 해결

## 🔢 기본 구성 요소

### 스칼라 (Scalar)
크기만 있고 방향이 없는 값
- 예: 질량, 온도, 속력
- 단순한 실수 또는 복소수

### 벡터 (Vector)
크기와 방향을 동시에 나타내는 수학적 객체
- 예: 속도, 힘, 변위
- n차원 공간의 점이나 방향을 표현

**벡터의 종류:**
- **공간벡터**: 출발점과 도착점이 있는 벡터
- **위치벡터**: 원점을 기준으로 한 벡터 (계산에 주로 사용)

### 행렬 (Matrix)
벡터를 원소로 가지는 2차원 배열
- 선형변환의 표현
- 연립방정식의 계수 저장
- 로봇의 기하학적 변환 표현

### 텐서 (Tensor)
고차원 배열로 확장된 개념
- 스칼라(0차), 벡터(1차), 행렬(2차)의 일반화
- 복잡한 물리적 관계 표현

## 🔧 기본 연산

### 벡터 연산
```
벡터 덧셈: u + v = [u₁+v₁, u₂+v₂, ..., uₙ+vₙ]
스칼라 곱: ku = [ku₁, ku₂, ..., kuₙ]
내적(Inner Product): u·v = u₁v₁ + u₂v₂ + ... + uₙvₙ
외적(Cross Product): u×v (3차원에서만 정의)
```

### 벡터 노름 (Norm)
**L1 노름:** ||x||₁ = |x₁| + |x₂| + ... + |xₙ| (맨하탄 거리)
**L2 노름:** ||x||₂ = √(x₁² + x₂² + ... + xₙ²) (유클리드 거리)

### 행렬 연산
```
행렬 덧셈: A + B (같은 크기 행렬만 가능)
행렬 곱셈: AB (A의 열 수 = B의 행 수)
전치행렬: Aᵀ (행과 열 교환)
역행렬: A⁻¹ (AA⁻¹ = I)
```

## 🤖 로봇공학에서의 응용

### 좌표 변환
```
동차변환행렬: T = [R  p]
                  [0  1]
- R: 3×3 회전행렬
- p: 3×1 위치벡터
```

### 운동학 해석
- **순기구학**: 관절각도 → 끝점 위치
- **역기구학**: 끝점 위치 → 관절각도
- **자코비안**: 관절공간과 작업공간의 속도 관계

### 동역학 방정식
```
M(q)q̈ + C(q,q̇)q̇ + G(q) = τ
- M: 관성행렬
- C: 코리올리스/원심력 항
- G: 중력항
- τ: 토크벡터
```

### 제어 시스템
```
상태공간 표현: ẋ = Ax + Bu
                y = Cx + Du
- A: 시스템 행렬
- B: 입력 행렬
- C: 출력 행렬
- D: 전달 행렬
```

## 🧮 고급 개념

### 고유값과 고유벡터
```
Av = λv
- A: 행렬
- v: 고유벡터
- λ: 고유값
```

**로봇공학 응용:**
- 시스템 안정성 해석
- 주성분 분석(PCA)
- 진동 모드 해석

### 특이값 분해 (SVD)
```
A = UΣVᵀ
- U, V: 직교행렬
- Σ: 대각행렬 (특이값)
```

**응용:**
- 의사역행렬 계산
- 데이터 압축
- 로봇 제어의 최적화

### 무어-펜로즈 역행렬
정방행렬이 아니거나 특이행렬일 때 사용하는 일반화된 역행렬
```
A⁺ = VΣ⁺Uᵀ (SVD 이용)
```

**로봇공학 응용:**
- 과구동/소구동 시스템 해결
- 최소제곱해 계산
