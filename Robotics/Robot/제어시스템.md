# 제어시스템

> 상위: [[제어]]

로봇의 궤적 계획부터 고급 제어 알고리즘까지 제어 이론을 다룹니다.

## 📚 하위 영역

### [[궤적계획]]
로봇의 경로와 시간 계획을 다루는 분야

### [[기본제어]]
PID 제어 등 기초적인 피드백 제어 방법들

### [[고급제어]]
적응제어, 견고제어 등 고급 제어 기법들

### [[최적제어]]
성능 지표를 최적화하는 제어 방법들

### [[ROS2]]
로봇 운영체제 2.0 - 분산 시스템을 위한 로봇 미들웨어

---

## 🎯 제어의 계층

### 계획 → 제어 → 최적화
1. 원하는 운동 계획
2. 계획 추종 제어
3. 성능 최적화

### 제어 공간
- **관절공간**: 관절 좌표에서의 제어
- **작업공간**: 작업 좌표에서의 제어
- **하이브리드**: 혼합 제어 방식
