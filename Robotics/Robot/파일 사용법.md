금속 부품의 표면 마감과 디버링(burr 제거)을 위한 파일의 올바른 사용법과 선택 기준을 다룹니다. 기본적인 파일링 기법부터 정밀한 형상 가공까지 로봇 제작에서 필요한 핵심 기술들을 체계적으로 설명합니다.

## 🔧 파일의 종류와 특징

### 거칠기별 분류
1. **거친 파일 (Coarse Cut)**
   - 날 간격: 3-6 TPI (Teeth Per Inch)
   - 용도: 거친 형상 가공, 대량 재료 제거
   - 재료: 연강, 알루미늄, 황동

2. **중간 파일 (Bastard Cut)**
   - 날 간격: 8-12 TPI
   - 용도: 일반적인 형상 가공
   - 균형 잡힌 절삭량과 표면 조도

3. **세밀 파일 (Smooth Cut)**
   - 날 간격: 14-20 TPI
   - 용도: 정밀 가공, 표면 마감
   - 우수한 표면 조도 달성

4. **초세밀 파일 (Dead Smooth)**
   - 날 간격: 20+ TPI
   - 용도: 최종 마감, 정밀 피팅
   - 거울면에 가까운 표면 조도

### 형태별 분류
1. **평파일 (Flat File)**
   - 한쪽 또는 양쪽 편평면
   - 직선 모서리와 평면 가공용
   - 크기: 100mm~350mm

2. **반원파일 (Half Round File)**
   - 한쪽은 평면, 한쪽은 곡면
   - 내부 곡선과 홈 가공용
   - 다양한 반지름 대응

3. **원파일 (Round File)**
   - 원형 단면
   - 구멍 확대와 내부 곡면 가공
   - 직경: 3mm~25mm

4. **각파일 (Square File)**
   - 정사각형 단면
   - 사각 구멍과 홈 가공용
   - 직각 내각 가공에 최적

5. **삼각파일 (Triangular File)**
   - 삼각형 단면
   - 예각부와 V홈 가공용
   - 톱니 정비용으로도 활용

6. **니들파일 (Needle File)**
   - 소형 정밀 파일
   - 미세 작업과 정밀 가공용
   - 전자 부품 피팅 작업

### 날 패턴별 분류
1. **싱글 컷 (Single Cut)**
   - 한 방향으로만 날이 배열
   - 부드러운 절삭과 우수한 표면 조도
   - 정밀 가공과 마감 작업용

2. **더블 컷 (Double Cut)**
   - 두 방향 교차 날 배열
   - 빠른 재료 제거 가능
   - 거친 가공과 형상 작업용

3. **커브드 컷 (Curved Cut)**
   - 곡선 날 패턴
   - 부드러운 절삭감
   - 연질 재료 가공 최적화

## ⚙️ 기본 사용 방법

### 파일 선택
1. **재료별 선택**
   - 연강/탄소강: 일반 스틸 파일
   - 스테인리스강: 고급 스틸 또는 초경 파일
   - 알루미늄: 알루미늄 전용 파일 (칩 배출 최적화)
   - 황동: 세밀한 날의 파일

2. **작업 목적별 선택**
   - 거친 가공: 거친~중간 파일
   - 정밀 가공: 세밀~초세밀 파일
   - 디버링: 세밀 파일 또는 니들 파일
   - 표면 마감: 초세밀 파일

### 파일링 기법
1. **기본 자세**
   - 작업물을 바이스에 확실히 고정
   - 파일 핸들을 한 손으로 잡고, 끝을 다른 손으로 지지
   - 안정된 자세로 일정한 압력 유지

2. **스트로크 동작**
   - 앞으로 밀 때만 절삭 (푸시 스트로크)
   - 뒤로 당길 때는 압력 해제 (리턴 스트로크)
   - 전체 파일 길이를 활용한 긴 스트로크

3. **압력과 각도**
   - 적절한 압력으로 일정한 절삭 깊이 유지
   - 과도한 압력으로 인한 파일 파손 방지
   - 작업면에 대해 적절한 각도 유지

## 🎯 고급 파일링 기법

### 정밀 피팅 작업
1. **단계적 가공**
   - 거친 파일로 대략적 형상 가공
   - 중간 파일로 치수 근접 가공
   - 세밀 파일로 최종 치수 달성

2. **측정과 확인**
   - 자주 측정하여 과가공 방지
   - 버니어 캘리퍼스로 정확한 치수 확인
   - 시행착오를 통한 점진적 접근

### 표면 마감 기법
1. **교차 파일링**
   - 첫 번째 방향으로 거친 가공
   - 90도 방향으로 중간 가공
   - 최종 방향으로 세밀 마감

2. **점진적 마감**
   - 거친 → 중간 → 세밀 → 초세밀 순서
   - 각 단계에서 이전 가공 자국 완전 제거
   - 최종 단계에서 균일한 표면 조도 달성

### 특수 형상 가공
1. **곡선 가공**
   - 반원파일로 내부 곡선 가공
   - 원파일로 구멍 확대와 곡면 가공
   - 템플릿 사용으로 정확한 형상 확보

2. **각도 가공**
   - 삼각파일로 V홈과 예각부 가공
   - 평파일과 각파일 조합으로 복합 각도
   - 각도기로 정확도 검증

## 📊 가공 조건 가이드

### 재료별 파일링 조건
| 재료 | 권장 파일 | 압력 | 속도 | 주의사항 |
|------|-----------|------|------|----------|
| 연강 | 중간~세밀 | 보통 | 보통 | 칩 제거 주의 |
| 스테인리스강 | 세밀 컷 | 강함 | 느림 | 작업 경화 주의 |
| 알루미늄 | 알루미늄 전용 | 약함 | 빠름 | 칩 막힘 방지 |
| 황동 | 세밀~초세밀 | 약함 | 보통 | 부드러운 가공 |
| 플라스틱 | 초세밀 | 매우 약함 | 느림 | 용융 방지 |

### 표면 조도 달성 기준
- **거친 파일**: Ra 6.3-12.5 μm (거친 가공)
- **중간 파일**: Ra 3.2-6.3 μm (일반 가공)
- **세밀 파일**: Ra 1.6-3.2 μm (정밀 가공)
- **초세밀 파일**: Ra 0.8-1.6 μm (마감 가공)

## ⚠️ 안전 수칙

### 작업 안전
1. **개인 보호 장비**
   - 안전안경 착용 (금속 칩 비산 방지)
   - 작업용 장갑으로 손 보호
   - 적절한 조명으로 정확한 작업

2. **작업 환경**
   - 안정된 작업대와 바이스 사용
   - 충분한 작업 공간 확보
   - 금속 칩 정리를 위한 준비

### 파일 안전
1. **적절한 사용**
   - 파일을 해머나 지렛대로 사용 금지
   - 과도한 압력으로 인한 파손 방지
   - 날 방향과 반대로 사용 금지

2. **핸들 관리**
   - 반드시 핸들 장착 후 사용
   - 느슨한 핸들 즉시 교체
   - 파일 탱(tang) 노출 상태로 사용 금지

## 🔧 유지보수 및 관리

### 일상 관리
1. **파일 청소**
   - 파일 카드(file card)로 날 사이 칩 제거
   - 브러시로 정기적인 청소
   - 필요시 케로신이나 솔벤트 사용

2. **보관 방법**
   - 개별 보관으로 날 손상 방지
   - 습기 방지를 위한 건조한 환경
   - 보호 캡이나 케이스 사용

### 날 관리
1. **날카로움 유지**
   - 적절한 압력으로 날 마모 최소화
   - 한 방향 사용으로 날 보호
   - 경화된 재료 가공 시 주의

2. **수명 연장**
   - 작업 전 청소로 이물질 제거
   - 적절한 윤활으로 마찰 감소
   - 올바른 보관으로 부식 방지

### 교체 기준
1. **날 마모**
   - 절삭력 현저히 감소
   - 날끝 둥글어짐 또는 마모
   - 원하는 표면 조도 달성 불가

2. **구조적 손상**
   - 파일 몸체 균열 또는 변형
   - 핸들 결합부 손상
   - 과도한 부식이나 녹 발생

## 📋 실무 활용 가이드

### 로봇 제작 시 활용
- **디버링**: 절단이나 드릴링 후 날카로운 모서리 제거
- **피팅 작업**: 베어링이나 부싱의 정밀한 끼워맞춤
- **표면 마감**: 용접부나 가공면의 매끄러운 마감
- **형상 수정**: 설계 변경이나 오차 보정을 위한 미세 가공

### 품질 향상 팁
1. **측정과 가공의 반복**
   - 자주 측정하여 과가공 방지
   - 점진적 접근으로 정확도 향상
   - 템플릿이나 게이지 활용

2. **일관된 기법**
   - 동일한 압력과 각도 유지
   - 규칙적인 스트로크 패턴
   - 작업자 간 기법 표준화

### 효율성 개선
1. **단계적 가공 계획**
   - 거친 가공부터 마감까지 체계적 접근
   - 적절한 파일 순서와 조합
   - 불필요한 작업 최소화

2. **도구 준비**
   - 다양한 크기와 형태의 파일 세트
   - 파일 카드와 청소 도구
   - 측정 도구와 검증 장비
