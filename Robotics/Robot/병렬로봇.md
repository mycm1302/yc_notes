## 🎯 정의 및 특징

**병렬로봇**은 닫힌 루프 기구학 구조를 가진 로봇으로, 여러 개의 독립적인 체인이 움직이는 플랫폼과 고정 베이스를 연결하는 구조입니다.

### 🏗️ 구조적 특징
```
고정 베이스 → 다중 체인 → 움직이는 플랫폼
- 체인 수: 통상 3-6개
- 자유도: 3-6 DOF
- 구조: 닫힌 운동학적 체인
```

### ⚡ 장점
- **고정밀도**: 반복 정확도 ±0.02mm 이하
- **고강성**: 다중 지지로 인한 구조적 강성
- **고하중**: 여러 액추에이터의 하중 분산
- **고속도**: 가벼운 움직이는 부분

### ⚠️ 단점
- **복잡한 기구학**: 순기구학 해가 복잡
- **제한된 작업공간**: 특이점으로 인한 제약
- **높은 비용**: 정밀 제조 요구

## 🔧 기술적 기초

### 1. 구조 합성 (Structural Synthesis)
**기구학적 구조 설계**
- **이동 플랫폼**: 3-6 자유도 설정
- **연결 체인**: 각 체인의 관절 구성
- **액추에이터**: 능동/수동 관절 배치

### 2. 기구학 (Kinematics)

#### 역기구학 (Inverse Kinematics)
```
주어진: 플랫폼 위치/자세 (x, y, z, α, β, γ)
구하는: 액추에이터 변위 (q₁, q₂, ..., qₙ)

일반적으로 해석적 해 존재
```

#### 순기구학 (Forward Kinematics)
```
주어진: 액추에이터 변위 (q₁, q₂, ..., qₙ)
구하는: 플랫폼 위치/자세 (x, y, z, α, β, γ)

복잡한 비선형 방정식 → 수치해법 필요
```

### 3. 자코비안 분석

#### 속도 관계식
```
ẋ = J⁻¹ q̇
```
- **J**: 자코비안 행렬 (6×6)
- **ẋ**: 플랫폼 속도
- **q̇**: 액추에이터 속도

#### 정적 관계식
```
τ = Jᵀ F
```
- **τ**: 액추에이터 힘
- **F**: 플랫폼에 작용하는 힘

### 4. 특이점 분석 (Singularity Analysis)

#### 특이점 유형
1. **Type I 특이점**: det(J) = 0
   - 무한 미소 운동 가능
   - 제어성 상실

2. **Type II 특이점**: det(Jq) = 0  
   - 내부 응력 발생 가능
   - 구조적 불안정

3. **Type III 특이점**: Type I + Type II
   - 가장 위험한 상황

### 5. 작업공간 분석 (Workspace Analysis)

#### 작업공간 유형
- **도달 가능 작업공간**: 물리적 제약 고려
- **정향 작업공간**: 모든 자세 가능 영역
- **특이점 없는 작업공간**: 안전 작업 영역

#### 분석 방법
- **기하학적 방법**: CAD 기반 시각화
- **수치적 방법**: 격자점 샘플링
- **해석적 방법**: 경계면 방정식

### 6. 동역학 (Dynamics)

#### 운동 방정식
```
M(q)q̈ + C(q,q̇)q̇ + G(q) = τ
```
- **M(q)**: 관성 행렬
- **C(q,q̇)**: 코리올리/원심력 행렬
- **G(q)**: 중력 벡터
- **τ**: 액추에이터 토크

#### 뉴턴-오일러 방법
- 각 강체의 운동 방정식
- 제약력 해석적 계산

#### 라그랑지 방법
- 에너지 기반 접근
- 제약 조건 자동 처리

### 7. 강성 분석 (Stiffness Analysis)

#### 강성 행렬
```
δF = K δx
```
- **K**: 6×6 강성 행렬
- **δF**: 힘 변화
- **δx**: 위치 변화

#### 강성 최적화
- **등방성 강성**: 모든 방향 균등
- **선택적 강성**: 작업 특성에 맞춤

## 🔧 주요 구조 유형

### 1. Stewart Platform
- **구조**: 6-UPS (Universal-Prismatic-Spherical)
- **자유도**: 6 DOF
- **특징**: 가장 일반적인 구조

### 2. Delta Robot
- **구조**: 3-RRR (Revolute-Revolute-Revolute)
- **자유도**: 3 DOF (이동만)
- **특징**: 초고속 픽앤플레이스

### 3. Gough Platform
- **구조**: 6개 가변길이 다리
- **자유도**: 6 DOF
- **특징**: Stewart Platform의 원형
