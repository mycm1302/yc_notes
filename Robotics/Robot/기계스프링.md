# 기계스프링

> 상위: [[기계요소]]
> 
> 레퍼런스: Shigley's Mechanical Engineering Design Ch.10 "Mechanical Springs"

압축, 인장, 비틀림 스프링의 설계와 응력 계산을 다룹니다.

## 🌀 스프링의 기본 개념

### 스프링의 정의와 기능
**스프링 (Spring):**
- 탄성 변형을 이용해 에너지를 저장하고 방출하는 기계요소
- 충격 완화, 진동 감쇠, 복원력 제공

**주요 기능:**
```
1. 에너지 저장: E = (1/2)kx²
2. 하중 측정: 스프링 밸런스
3. 충격 흡수: 자동차 현가장치
4. 복원력: 접점 압력 유지
5. 진동 제어: 댐핑과 조합

k: 스프링 상수 (N/m)
x: 변위 (m)
```

### 스프링의 분류
```
형상별:
- 코일 스프링: 원통형, 원뿔형, 배럴형
- 판 스프링: 겹판, 단판, 접시
- 비틀림 막대: 토션 바
- 선형 스프링: 외팔보, 단순보

하중별:
- 압축 스프링: 축방향 압축
- 인장 스프링: 축방향 인장  
- 비틀림 스프링: 회전 모멘트
- 굽힘 스프링: 굽힘 모멘트
```

## 🔄 원통형 코일 스프링

### 기본 형상과 용어
```
기본 치수:
- d: 와이어 직경 (Wire diameter)
- D: 코일 평균 직경 (Mean coil diameter)
- Do: 외경 = D + d
- Di: 내경 = D - d
- p: 피치 (Pitch)
- L0: 자유 길이 (Free length)

권수:
- Nt: 총 권수 (Total coils)
- Na: 유효 권수 (Active coils)
- Ne: 단부 권수 (End coils)
```

### 스프링 지수와 기하학
```
스프링 지수: C = D/d

권장값: C = 4~12
- C < 4: 제조 어려움, 높은 응력
- C > 12: 좌굴 위험, 탄성 불안정

코일 길이: Ls = Na × d (견고 길이)
피치각: α = tan⁻¹(p/(πD))
```

## 💪 압축 스프링 설계

### 전단응력과 수정계수
```
기본 전단응력: τ = 8FD/(πd³)

Wahl 곡률보정계수:
Ks = (4C-1)/(4C-4) + 0.615/C

수정 전단응력: τ = Ks × 8FD/(πd³)

F: 하중 (N)
D: 코일 평균직경 (m)
d: 와이어 직경 (m)
```

### 스프링 상수
```
스프링 상수: k = Gd⁴/(8D³Na)

G: 전단탄성계수 (Pa)
Na: 유효 권수

단위 환산:
- SI 단위: N/m
- 실용 단위: kgf/mm, N/mm
```

### 변위와 하중 관계
```
후크의 법칙: F = kx

최대 하중시:
- 작동 하중: F1 (설계 하중)
- 압축 변위: x1 = F1/k
- 압축 길이: L1 = L0 - x1

견고시 (Solid height):
- 견고 길이: Ls = (Nt + 1) × d
- 견고 하중: Fs = k × (L0 - Ls)
```

## 📊 스프링 재료와 허용응력

### 스프링강 종류
**탄소강:**
```
경질강선 (Hard drawn):
- 탄소함량: 0.6~0.9%
- 직경: 1~12mm
- 인장강도: 1600~2200 MPa
- 용도: 일반용 스프링

오일 템퍼드 강선 (Oil tempered):
- 담금질-뜨임 처리
- 직경: 3~60mm  
- 인장강도: 1400~1900 MPa
- 피로 특성 우수
```

**합금강:**
```
크롬-바나듐강 (Cr-V):
- 고온 특성 우수
- 직경: 6~60mm
- 인장강도: 1500~1900 MPa
- 용도: 자동차 현가, 밸브

크롬-실리콘강 (Cr-Si):
- 내피로성 최우수
- 직경: 3~16mm
- 인장강도: 1700~2100 MPa
- 용도: 고응력 스프링
```

**스테인리스강:**
```
SUS304 (18-8):
- 인장강도: 1600 MPa
- 내식성 우수
- 비자성
- 용도: 화학장치, 식품기계

SUS316 (18-8-3):
- 내해수성 우수
- 인장강도: 1500 MPa
- 용도: 해양 구조물
```

### 허용응력과 안전계수
```
허용 전단응력:
τall = Ssy/n

Ssy: 전단 항복강도 ≈ 0.5 × Sut
Sut: 인장강도
n: 안전계수

안전계수:
- 정적 하중: n = 1.2~1.5
- 반복 하중: n = 1.5~2.0
- 충격 하중: n = 2.0~3.0
```

## 🔄 압축 스프링 설계 절차

### 1단계: 기본 사양 결정
```
주어진 조건:
- 최대 하중: Fmax
- 허용 변위: xmax
- 공간 제약: 외경, 길이

스프링 상수: k = Fmax/xmax
```

### 2단계: 와이어 직경 결정
```
응력 조건에서:
d ≥ ∛(8FmaxD × Ks/(π × τall))

강성 조건에서:
d ≥ ∜(8FmaxD³/(π × G × xmax))

두 조건을 만족하는 d 선택
```

### 3단계: 유효 권수 계산
```
Na = Gd⁴/(8D³k)

정수로 반올림하여 실제 권수 결정
실제 스프링 상수: k' = Gd⁴/(8D³Na)
```

### 4단계: 자유 길이 결정
```
L0 = Ls + xmax + 여유

여유 = 5~30% × xmax
좌굴 방지와 응력 완화 고려
```

## 📏 스프링 안정성 설계

### 좌굴 (Buckling)
```
좌굴 조건: L0/D > 2.6

좌굴 방지 방법:
1. L0/D < 2.6 유지
2. 내부 가이드 로드
3. 외부 슬리브
4. 양단 고정

임계 하중: Fcr = π²EI/(μL²)
μ: 단부 조건 계수
```

### 횡진동 (Surging)
```
고유진동수: fn = (1/2π)√(k/m)

m: 스프링 질량 ≈ (1/3) × 총질량

진동 방지:
- 작동진동수 < fn/3
- 가변 피치 적용
- 댐핑 추가
```

### 공진 방지
```
1차 고유진동수: f1 = (d/4D²)√(G/ρ)/Na

ρ: 밀도 (kg/m³)
G: 전단탄성계수 (Pa)

권장 조건: f작동 < f1/13
```

## ↗️ 인장 스프링 설계

### 인장 스프링 특성
```
구조:
- 양단에 고리 (Loop) 또는 갈고리 (Hook)
- 초기 장력 존재 (Fi)
- 밀착 권선 (Close-wound)

하중-변위 관계:
F = Fi + kx (x ≥ 0)
```

### 초기 장력
```
초기 전단응력: τi = 8FiD/(πd³)

권장값: τi = 0.15~0.5 × Ssy

초기 장력 공식:
Fi = (A × Ssy/Ks) × (πd³)/(8D)

A: 경험계수 (0.15~0.5)
```

### 고리와 갈고리 응력
```
고리 응력: σ = 16FR/(πd³) + 4F/(πd²)

R: 고리 평균 반지름
첫 번째 항: 굽힘 응력
두 번째 항: 인장 응력

갈고리 응력 더 높음: 곡률반지름 작기 때문
```

## 🔄 비틀림 스프링 설계

### 비틀림 스프링 구조
```
형태: 원통형 코일, 양단에 암(Arm)
하중: 순수 토크 또는 굽힘 모멘트
변형: 코일 직경 변화 + 각변위

용도: 클로저, 힌지, 카운터밸런스
```

### 응력과 변형
```
굽힘 응력: σ = 32M/(πd³) × Ki

각변위: θ = 64MR²Na/(Ed⁴) (라디안)

M: 굽힘 모멘트
Ki: 곡률보정계수 ≈ 1.2~1.3
R: 코일 반지름
E: 탄성계수
```

### 토크-각도 관계
```
토크 상수: kθ = M/θ = Ed⁴/(64R²Na)

에너지 저장: U = (1/2)kθθ²

단위: N·m/rad
```

## 📄 판스프링 설계

### 겹판 스프링 (Leaf Spring)
```
구조: 여러 매의 판을 겹쳐서 구성
특징: 높은 하중 용량, 진동 감쇠

주판 응력: σ = 6FL/(bnth²)

F: 하중, L: 길이, b: 폭
nt: 총 매수, h: 판 두께
```

### 접시 스프링 (Belleville Spring)
```
형상: 원뿔형 접시 모양
특징: 큰 하중, 작은 변위

하중-변위 관계: 비선형
용도: 볼트 예하중, 안전밸브
```

### 스프링 상수 (겹판)
```
k = 3Ebt³/(4L³) × (n+1)/n

n: 보조판 매수
주판보다 짧은 보조판들의 효과
```

## 🌊 동적 특성과 피로

### 스프링의 피로
```
피로 파괴: 표면에서 시작
응력 집중: 와이어 표면 결함
환경: 부식 피로 가속

응력비: R = σmin/σmax
완전 양진: R = 0
부분 양진: R > 0
```

### 피로 수명 설계
```
수정 굿맨 선도:
τa/Se + τm/Ssu ≤ 1

τa: 전단응력 진폭
τm: 평균 전단응력  
Se: 전단 피로한도
Ssu: 전단 인장강도

Se ≈ 310 MPa (고강도 스프링강)
```

### 피로 수명 향상
```
표면처리:
- 쇼트피닝: 압축잔류응력 도입
- 니켈 도금: 내식성 향상
- 인산피막: 윤활성 향상

설계 개선:
- 응력 집중 최소화
- 적절한 안전계수
- 온도 관리
```

## 🌡️ 스프링의 온도 특성

### 온도 보상
```
탄성계수 온도계수:
dG/dT ≈ -400 MPa/°C (강)

스프링 상수 변화:
dk/dT = k × (dG/dT)/G

일반적으로: 온도 상승시 강성 감소
```

### 고온용 스프링
```
재료:
- 인코넬 X-750: ~700°C
- 하스텔로이 C: ~800°C
- 세라믹: ~1000°C

설계 고려사항:
- 크리프 저항성
- 산화 저항성  
- 열팽창 보상
```

### 저온용 스프링
```
문제점:
- 취성 증가
- 탄성계수 증가
- 충격 저항성 감소

대책:
- 저온용 재료 선택
- 적절한 안전계수
- 충격 방지 설계
```

## 🔧 스프링 제조와 가공

### 냉간 성형 (Cold Forming)
```
적용: 직경 < 12mm
공정: 와이어 공급 → 성형 → 절단
장점: 생산성 우수, 정밀도 높음
단점: 잔류응력, 크기 제한
```

### 열간 성형 (Hot Forming)
```
적용: 직경 > 12mm  
온도: 900~1100°C
공정: 가열 → 성형 → 담금질 → 뜨임
장점: 대형 스프링 제작 가능
단점: 치수 정밀도 낮음
```

### 후처리 공정
```
응력 제거 (Stress Relieving):
- 온도: 400~450°C
- 시간: 30~60분
- 목적: 잔류응력 제거

쇼트피닝 (Shot Peening):
- 강구로 표면 타격
- 압축잔류응력 도입
- 피로 강도 30~50% 향상

세팅 (Setting):
- 105~110% 하중으로 압축
- 영구변형으로 잔류응력 완화
```

## 📏 스프링 측정과 시험

### 치수 측정
```
측정 항목:
- 자유 길이: L0
- 외경: Do  
- 와이어 직경: d
- 권수: Nt, Na
- 수직도: 기울어짐 각도

허용차:
- 길이: ±1~3%
- 직경: ±2~5%
- 하중: ±5~10%
```

### 하중 시험
```
압축 시험:
- 지정 길이에서 하중 측정
- 스프링 상수 계산
- 세팅 하중 확인

내구 시험:
- 반복 하중 시험
- 10⁶~10⁷ 사이클
- 파괴 또는 하중 감소 확인
```

### 동특성 시험
```
고유진동수 측정:
- 가진기로 주파수 스윕
- 공진점 검출
- 감쇠비 측정

충격 응답:
- 충격 하중 인가
- 진동 감쇠 특성
- 동적 증폭 계수
```

## 🎯 특수 스프링 시스템

### 가스 스프링
```
구조: 실린더 + 피스톤 + 고압 가스
특성:
- 일정한 힘 (등압 과정)
- 부드러운 작동
- 댐핑 기능 내장

용도: 자동차 트렁크, 의자
```

### 공기 스프링
```
구조: 고무 벨로우즈 + 압축공기
특성:
- 높이 조절 가능
- 진동 절연 우수
- 높은 하중 용량

용도: 버스 현가, 정밀기계 방진
```

### 자기 스프링
```
구조: 영구자석 또는 전자석
특성:
- 비접촉 지지
- 강성 가변 가능
- 무마찰

용도: 자기 베어링, 액추에이터
```

## 💰 스프링 경제성과 표준화

### 표준 스프링 활용
```
장점:
- 낮은 단가
- 짧은 납기
- 검증된 품질
- 쉬운 교체

제약:
- 치수 제한
- 성능 타협
- 설계 자유도 제한
```

### 커스텀 스프링
```
고려사항:
- 개발비용: 금형, 치공구
- 최소 주문량
- 납기: 4~8주
- 시험 검증 비용

경제성 분석:
Break-even = 개발비/(표준품 단가 - 커스텀 단가)
```

### 라이프사이클 비용
```
LCC = 초기비용 + 운영비용 + 교체비용

고성능 스프링:
- 높은 초기비용
- 긴 수명
- 낮은 유지비용

표준 스프링:
- 낮은 초기비용  
- 짧은 수명
- 빈번한 교체
```
