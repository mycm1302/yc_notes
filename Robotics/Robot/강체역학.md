# 강체역학

> 상위: [[물리적기초]]

강체의 운동과 동역학을 다루는 전문 분야로, 로봇 링크의 운동 해석에 핵심적입니다.

## 🔄 강체 운동

### 기본 운동
- [[강체운동]] - 강체의 기본적인 운동 특성
- [[강체동역학]] - 강체에 작용하는 힘과 운동의 관계

### 관성 특성
- [[관성텐서]] - 질량 분포와 회전 관성
- [[각운동량]] - 회전 운동량과 보존 법칙

## 🎯 기하학적 변환

### 좌표 변환
- [[유클리드변환]] - 회전과 병진의 결합

## 🔗 로봇공학 응용

### 기구학 연결
- 강체 운동 → 로봇 링크 운동
- 좌표 변환 → 기구학 체인

### 동역학 연결
- 관성 텐서 → 로봇 관성 행렬
- 각운동량 → 회전 관절 동역학

### 제어 연결
- 강체 동역학 → 동역학 기반 제어
- 관성 특성 → 적응 제어
