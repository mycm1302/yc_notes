# 토크렌치 사용법

> 상위: [[설계도구]]
> 
> 레퍼런스: 
> - **ISO 6789-1:2017 & 6789-2:2017** - 토크 렌치 설계, 품질 및 교정 국제 표준
> - **Constellation PowerLabs Calibration Guide** - 토크 렌치 교정 및 성능 검증 절차
> - **Equipment World Maintenance Manual** - 토크 렌치 현장 교정 단계별 가이드
> - **Mountz Torque Application Engineering** - 토크 제어 도구 교정 표준 체계

로봇 조립에서 정확한 체결 토크를 보장하기 위한 토크렌치의 올바른 사용법과 교정 방법을 다룹니다. 안전하고 신뢰할 수 있는 체결을 위한 기본 원리부터 정밀 교정까지 실무에 필요한 모든 내용을 체계적으로 설명합니다.

## 🔧 토크렌치의 종류와 원리

### 주요 토크렌치 유형
1. **클릭형 토크렌치 (Click-Type)**
   - 설정 토크 도달 시 클릭 음으로 알림
   - 가장 일반적이고 정확한 방식
   - 토크 범위: 5-500 Nm (모델별 상이)
   
2. **다이얼형 토크렌치 (Dial-Type)**
   - 아날로그 게이지로 실시간 토크 표시
   - 연속적인 토크 모니터링 가능
   - 정밀 작업이나 교정용으로 활용

3. **디지털 토크렌치 (Digital)**
   - LCD 디스플레이로 정확한 수치 표시
   - 데이터 저장 및 분석 기능
   - 높은 정밀도와 다양한 단위 지원

4. **빔형 토크렌치 (Beam-Type)**
   - 구조가 단순하여 교정 빈도 낮음
   - 포인터와 스케일로 토크 측정
   - 경제적이지만 정밀도는 상대적으로 낮음

### 작동 원리 (클릭형 기준)
1. **스프링 장치**
   - 조정 가능한 스프링이 설정 토크 결정
   - 핸들 회전으로 스프링 압축력 조절
   
2. **해제 메커니즘**
   - 설정 토크 도달 시 베어링이 노치에서 이탈
   - 레버 암이 갑작스럽게 회전하며 클릭 소음 발생
   
3. **토크 전달**
   - 래칫 헤드를 통해 소켓에 토크 전달
   - 정확한 토크에서만 체결 완료

## ⚙️ 기본 사용 방법

### 사전 준비
1. **적절한 토크렌치 선택**
   - 요구 토크의 20-80% 범위 내 모델 선택
   - 토크 범위 상한의 80% 이하에서 사용 권장
   - 정밀도: 일반적으로 ±3-4% (ISO 6789 기준)

2. **토크 사양 확인**
   - 제조사 매뉴얼에서 체결 토크 확인
   - 재료별, 나사 크기별 표준 토크 참조
   - 윤활 조건에 따른 토크 보정 고려

3. **도구 점검**
   - 교정 유효 기간 확인 (6개월 또는 5,000 사이클)
   - 외관 손상이나 변형 여부 점검
   - 클릭 음의 명확성 사전 테스트

### 사용 절차
1. **토크 설정**
   - 핸들 끝의 잠금 노브 해제
   - 원하는 토크값으로 핸들 회전 조정
   - 설정 완료 후 잠금 노브 체결

2. **체결 작업**
   - 적절한 소켓 선택 및 장착
   - 볼트를 손으로 먼저 조임 (finger-tight)
   - 토크렌치로 천천히 일정한 속도로 조임
   - 클릭 소음 발생 즉시 작업 중단

3. **작업 완료 후**
   - 토크 설정을 최소값으로 조정 (스프링 이완)
   - 청결한 상태로 보관
   - 사용 기록 작성 (교정 관리용)

## 📊 토크 사양 가이드

### 일반적인 볼트 토크 (건조 상태)
| 나사 규격 | 재질 | 토크 (Nm) | 비고 |
|-----------|------|-----------|------|
| M3 | 스테인리스 | 1.2-1.8 | 소형 전자 부품 |
| M4 | 스테인리스 | 2.5-3.5 | 센서 마운팅 |
| M5 | 스테인리스 | 5-7 | 일반 조립 |
| M6 | 스테인리스 | 8-12 | 프레임 체결 |
| M8 | 스테인리스 | 20-25 | 메인 구조체 |
| M10 | 스테인리스 | 35-45 | 베어링 하우징 |

### 재료별 토크 보정 계수
- **건조 상태**: 1.0 (기준값)
- **윤활유 적용**: 0.7-0.8 (토크 감소)
- **스레드락 적용**: 1.1-1.2 (토크 증가)
- **부식 방지 코팅**: 0.9-1.0 (약간 감소)

## 🎯 정밀 체결 기법

### 다단계 체결법
1. **예비 체결**
   - 목표 토크의 30-50%로 1차 체결
   - 모든 볼트를 균등하게 예비 체결
   
2. **순차 체결**
   - 대각선 또는 별형 패턴으로 체결 순서 결정
   - 목표 토크의 75%로 2차 체결
   
3. **최종 체결**
   - 최종 토크로 마무리 체결
   - 필요시 재토크 확인 (settling 고려)

### 중요 체결부 관리
1. **임계 볼트 식별**
   - 안전과 직결되는 체결부 명시
   - 추가 검증 절차 수립
   
2. **토크 기록 관리**
   - 체결 토크값과 작업자 기록
   - 재점검 주기 설정 (진동 환경 고려)

## 🔄 교정 (Calibration)

### 교정 주기 (ISO 6789 기준)
- **일반 사용**: 6개월 또는 5,000 사이클
- **정밀 작업**: 3개월 또는 2,500 사이클  
- **가혹 환경**: 월 1회 또는 1,000 사이클
- **임계 용도**: 매주 또는 500 사이클

### 현장 간이 교정법
1. **필요 장비**
   - 정확한 저울 (20 pound 추 권장)
   - 줄자 또는 버니어 캘리퍼스
   - 고정용 바이스
   - 로프 (1.5-2m)

2. **교정 절차**
   ```
   교정 거리 측정: 스퀘어 드라이브 중심 → 손잡이 위치
   설정 토크 계산: 교정 거리(inch) × 추 무게(lbs) ÷ 12 = ft-lbs
   실제 토크 측정: 클릭 발생 지점에서 재계산
   오차 확인: (측정값 - 설정값) ÷ 설정값 × 100%
   ```

3. **허용 오차 기준**
   - ±4% 이내: 정상 (ISO 6789 기준)
   - ±4-6%: 주의 (재교정 검토)
   - ±6% 초과: 즉시 전문 교정 필요

### 전문 교정
1. **공인 교정기관 이용**
   - ISO/IEC 17025 인증 시험소
   - NIST 추적 가능한 교정 표준
   - 교정 성적서 발급 및 관리

2. **교정 후 관리**
   - 교정 스티커 부착 (차기 교정일 표시)
   - 교정 이력 데이터베이스 관리
   - 사용 중 정기적 기능 점검

## ⚠️ 안전 수칙

### 사용 중 안전 수칙
1. **올바른 사용법**
   - 토크렌치를 연장 도구로 사용 금지
   - 설정 토크를 초과하여 강제로 회전 금지
   - 토크 방향과 반대로 사용 금지 (역토크)

2. **신체 안전**
   - 갑작스러운 클릭으로 인한 손목 부상 주의
   - 안정된 자세에서 일정한 속도로 조작
   - 무리한 힘 사용 금지

### 장비 안전
1. **적절한 보관**
   - 사용 후 토크 설정을 최소값으로 조정
   - 충격이나 낙하 방지
   - 습기와 부식 방지 환경에서 보관

2. **정기 점검**
   - 클릭 메커니즘의 정상 작동 확인
   - 외관 손상이나 변형 점검
   - 핸들과 헤드 결합부 이상 점검

---

## 🛠️ 유지보수

### 일상 관리
1. **청소와 윤활**
   - 사용 후 외부 이물질 제거
   - 회전 부품에 적절한 윤활 (제조사 권장)
   - 과도한 윤활로 인한 정확도 저하 주의

2. **기능 점검**
   - 정기적인 클릭 기능 확인
   - 토크 설정 메커니즘 원활성 점검
   - 래칫 헤드 작동 상태 확인

### 고장 진단
1. **클릭 음 미발생**
   - 스프링 장력 부족 (조정 필요)
   - 메커니즘 내부 마모 (전문 수리)
   
2. **클릭 음 조기 발생**
   - 스프링 장력 과다 (조정 필요)
   - 교정 오차 (재교정 필요)

3. **불규칙한 클릭**
   - 내부 오염이나 마모
   - 즉시 사용 중단 후 점검

## 📋 실무 활용 가이드

### 로봇 조립 시 활용
- **프레임 체결**: 구조적 안정성 확보를 위한 정밀 토크
- **모터 마운팅**: 진동 저항성과 정렬 정확도 보장  
- **베어링 하우징**: 예압 조절과 수명 최적화
- **센서 설치**: 위치 정확도와 내구성 확보

### 품질 관리 체계
1. **체결 기록 관리**
   - 볼트별 체결 토크와 작업자 기록
   - 재점검 일정과 결과 문서화
   
2. **교정 이력 관리**
   - 각 토크렌치별 교정 이력 데이터베이스
   - 사용 빈도와 교정 주기 최적화

3. **불량 예방**
   - 토크 부족으로 인한 체결 불량 방지
   - 과도한 토크로 인한 나사산 손상 방지
