## ⚙️ 기어의 기본 개념

### 기어의 정의와 기능
**기어 (Gear):**
- 회전축 사이에서 동력과 운동을 전달하는 기계요소
- 원주상에 배치된 치(齒)를 이용한 강제 맞물림
- 정확한 속도비와 높은 전달 효율

**주요 기능:**
```
1. 속도 변환: 증속 또는 감속
2. 토크 변환: 토크 증대 또는 감소  
3. 동력 전달: 회전 동력의 전달
4. 방향 변경: 회전축 방향 변경
5. 운동 분배: 하나의 입력을 여러 출력으로
```

### 기어의 장점과 단점
**장점:**
- **정확한 속도비**: 슬립 없는 강제 전달
- **높은 효율**: 98~99% (직접 맞물림)
- **큰 동력 전달**: 소형으로도 대용량 가능
- **내구성**: 적절한 설계시 장수명
- **정밀 제어**: 정확한 위치와 속도 제어

**단점:**
- **소음과 진동**: 치 맞물림에 의한 충격
- **정밀한 가공**: 높은 제조 정밀도 요구
- **윤활 필요**: 마모와 발열 방지
- **축간 거리 고정**: 중심거리 변경 어려움

## 🔧 기어의 분류

### 축 배치에 따른 분류
**평행축 기어:**
- **스퍼기어 (Spur Gear)**: 직선 치형, 가장 기본적
- **헬리컬기어 (Helical Gear)**: 경사 치형, 조용한 작동
- **더블 헬리컬기어**: 헤링본 형태, 축방향 하중 상쇄

**교차축 기어:**
- **베벨기어 (Bevel Gear)**: 원뿔형, 90° 교차가 일반적  
- **스파이럴 베벨기어**: 곡선 치형 베벨기어
- **하이포이드기어**: 축이 엇갈린 베벨기어

**엇갈린축 기어:**
- **웜기어 (Worm Gear)**: 큰 감속비, 자동잠금 가능
- **스크류기어**: 교차각이 임의인 헬리컬기어

### 치형 곡선에 따른 분류
**인볼루트 기어:**
- 가장 일반적인 치형
- 제조 오차에 대한 허용성 우수
- 중심거리 변화에 둔감

**사이클로이드 기어:**
- 고정밀 응용 (시계, 측정기)
- 제조 정밀도 엄격
- 백래시 최소화 가능

### 용도에 따른 분류
**동력 전달용**: 자동차, 산업기계, 로봇
**정밀 전달용**: 공작기계, 측정기, 광학기기  
**특수 용도**: 차동기어, 유성기어, 내접기어

## 📐 인볼루트 치형 기하학

### 인볼루트 곡선의 특성
**인볼루트 (Involute) 정의:**
- 기저원 (Base Circle) 둘레에 감긴 실을 펼 때 실 끝이 그리는 곡선
- 기저원 반지름: rb = r cos φ (φ: 압력각)
- 모든 인볼루트 치형의 공통선은 기저원에 접함

**인볼루트의 장점:**
```
1. 중심거리 변화에 둔감: 속도비 일정 유지
2. 제조 오차 허용성: 약간의 치형 오차 허용
3. 가공의 용이성: 직선 공구로 가공 가능
4. 강도 특성 우수: 치근부 두께 확보 용이
```

### 기어 치형의 기본 용어
```
모듈 (Module, m): 피치원 지름을 치수로 나눈 값
m = D/z = p/π (mm)

피치 (Pitch, p): 인접 치간의 호장 거리
p = πm = πD/z (mm)

압력각 (Pressure Angle, φ): 치형 접촉각
표준값: 20° (일반적), 14.5° (구형), 25° (고하중용)

피치원 직경: D = mz (mm)
기저원 직경: Db = D cos φ = mz cos φ

치수 (Number of Teeth): z
잇끝원 직경: Da = D + 2m = m(z + 2)
치뿌리원 직경: Df = D - 2.5m = m(z - 2.5)
```

## ⚙️ 스퍼기어 설계

### 기어비와 중심거리
```
기어비 (Gear Ratio):
i = ω1/ω2 = z2/z1 = D2/D1

ω: 각속도 (rad/s)
z: 치수  
D: 피치원 직경

중심거리 (Center Distance):
C = (D1 + D2)/2 = m(z1 + z2)/2

표준 중심거리: C = m(z1 + z2)/2
수정 중심거리: C' = C + xm (전위계수 적용)
```

### 치형 수정 (Profile Shift)
**전위계수 (Profile Shift Coefficient): x**
```
양의 전위 (+x): 잇끝쪽으로 이동
- 치근 두께 증가 → 강도 향상
- 언더컷 방지
- 중심거리 증가

음의 전위 (-x): 치뿌리쪽으로 이동  
- 잇끝 두께 감소
- 중심거리 감소
- 간섭 위험 증가

표준 기어: x = 0
전위 기어: x ≠ 0
```

### 언더컷과 최소 치수
```
언더컷 발생 조건:
z < 2/(sin²φ) = 17 (φ = 20°일 때)

최소 치수:
- 표준 기어: zmin = 17
- 전위 기어: 더 작은 치수 가능

언더컷 방지 방법:
1. 전위 사용: x > 0
2. 압력각 증가: φ = 25°
3. 특수 공구 사용
```

## 💪 기어 강도 설계

### 굽힘 강도 (Lewis 공식)
```
Lewis 기본 공식:
σ = Ft/(F·m·Y)

Ft: 접선력 (N)
F: 치폭 (면폭, mm)  
m: 모듈 (mm)
Y: Lewis 치형계수

AGMA 수정 공식:
σ = Ft·Ka·Km·Ks·Kv/(F·m·Y·KL·KR)

Ka: 응용계수 (Application factor)
Km: 하중분배계수 (Load distribution factor)  
Ks: 크기계수 (Size factor)
Kv: 동하중계수 (Dynamic factor)
KL: 수명계수 (Life factor)
KR: 신뢰도계수 (Reliability factor)
```

### 접촉 강도 (Hertz 응력)
```
Hertz 접촉응력:
σc = Cp·√(Ft·Ka·Km·Ks·Kv·CF/(F·D·I))

Cp: 탄성계수 (Elastic coefficient)
CF: 표면상태계수 (Surface condition factor)
I: 기하계수 (Geometry factor)
D: 피니언 피치원 직경

강-강 접촉: Cp = 191 √MPa
경화강: 허용응력 1000~1500 MPa
침탄강: 허용응력 1200~1700 MPa
```

## 🌀 헬리컬기어 설계

### 헬리컬기어의 특성
**나선각 (Helix Angle): β**
```
장점:
- 점진적 치 맞물림 → 소음 감소
- 접촉비 증가 → 하중 분산  
- 부드러운 동력 전달
- 고속 운전 적합

단점:
- 축방향 추력 발생
- 제조 복잡성 증가
- 축 베어링 하중 증가
```

### 헬리컬기어 기하학
```
법선 모듘: mn (Normal Module)
축방향 모듈: ma = mn/cos β
원주방향 모듈: mt = mn/cos β

피치원 직경: D = mnz/cos β = mtz

법선 압력각: φn (보통 20°)  
축방향 압력각: φt = tan⁻¹(tan φn/cos β)

중심거리: C = mn(z1 + z2)/(2 cos β)

나선각 권장값:
- 일반용: β = 15~30°
- 고속용: β = 20~35°  
- 정밀용: β = 8~20°
```

### 축방향 추력
```
축방향 추력: Fa = Ft tan β

Ft: 접선력
균형 방법:
1. 더블 헬리컬 (헤링본)
2. 추력 베어링 사용
3. 반대 나선각 기어 조합
```

## 🔺 베벨기어 설계

### 베벨기어의 기본 개념
**원뿔형 기어:**
```
용도: 교차축 동력 전달 (보통 90°)
구조: 원뿔면상에 형성된 치형
특징: 축 교차각 자유롭게 설정 가능

축각 (Shaft Angle): Σ = γ1 + γ2
피치원뿔각: γ1, γ2
일반적: Σ = 90° (직각 교차)
```

### 스트레이트 베벨기어
```
기하학적 관계:
tan γ1 = sin Σ/(mG + cos Σ)
tan γ2 = sin Σ/(m/G + cos Σ)

mG: 기어비 = z2/z1
γ1 + γ2 = Σ (축각)

피치원뿔 거리: A0 = D1/(2 sin γ1) = D2/(2 sin γ2)

백콘 거리: Ab = A0 cos γ
치형 폭: F ≤ A0/3 (일반적)

등가 스퍼기어 치수:
zv1 = z1/cos γ1 (가상 치수)
zv2 = z2/cos γ2
```

### 스파이럴 베벨기어
```
특징:
- 곡선 치형 (나선각 β = 35° 일반적)
- 점진적 맞물림
- 소음과 진동 감소
- 높은 하중 전달 능력

제조: 글리슨 방식, 크링엘른베르크 방식
용도: 자동차 디퍼렌셜, 공작기계
```

## 🌀 웜기어 설계

### 웜기어의 특성
**웜 (Worm)과 웜휠 (Worm Wheel):**
```
장점:
- 큰 감속비: 10:1 ~ 100:1 (한 단)
- 정숙한 운전
- 자동잠금 가능 (리드각 < 마찰각)
- 컴팩트한 구조

단점:
- 낮은 효율: 30~90%
- 발열 문제
- 축방향 추력 큼
- 정밀한 조립 필요
```

### 웜기어 기하학
```
기어비: i = zw/zg (zw: 웜 치수, zg: 웜휠 치수)
웜 리드: L = π × m × zw
웜 리드각: λ = tan⁻¹(L/(π × dw))

중심거리: C = m(zw + zg)/2
웜 외경: dw = m(zw + 2)
웜휠 피치원 직경: Dg = m × zg

압력각: 일반적으로 20°
웜 나사각: 2α (보통 40°)
```

### 웜기어 효율과 발열
```
효율: η = tan λ/(tan(λ + ρ))
ρ: 마찰각 = tan⁻¹μ

자동잠금 조건: λ < ρ
역전 방지 (브레이크 기능)

발열량: P손실 = P입력 × (1 - η)
냉각 설계 필수: 방열핀, 오일 순환
```

```
S45C, S50C: 일반 기어용

```

- 담금질성 우수

SNCM440 (Ni-Cr-Mo강):
- 고강도, 고인성
- 침탄 후 경도: 58~62 HRC

SCr440 (Cr강):

## 💧 기어 윤활

### 윤활의 목적
```
1. 마찰 감소: 치면 마찰계수 최소화
2. 마모 방지: 금속 접촉 방지
3. 냉각: 발생 열량 제거
4. 부식 방지: 산화, 부식 억제
5. 이물질 제거: 마모 입자 배출
```

### 윤활 방식
**오일 윤활:**
```
침지 윤활 (Oil Bath):
- 기어가 오일에 부분 침지
- 속도: ~20 m/s
- 간단, 경제적

분사 윤활 (Oil Jet):
- 고압으로 치면에 분사
- 고속: >20 m/s
- 냉각 효과 우수

순환 윤활 (Circulating):
- 펌프로 강제 순환
- 필터링, 냉각 가능
- 정밀 제어
```

**그리스 윤활:**
```
특징:
- 밀봉 간단
- 유지보수 적음
- 저속, 경하중용

한계:
- 냉각 효과 제한
- 속도 제한: <5 m/s
- 교체 주기 관리 필요
```

## 🔊 기어 소음과 진동

### 소음 발생 원인
```
1. 치형 오차: 피치 오차, 치형 오차
2. 맞물림 충격: 치끝 간섭, 급격한 하중 변화
3. 공진: 고유진동수와 맞물림 주파수 일치
4. 불평형: 편심, 중량 불균형
5. 윤활 불량: 경계 윤활, 오일 부족
```

### 소음 저감 방법
**설계적 대책:**
```
1. 헬리컬기어 사용: 점진적 맞물림
2. 전위 기어: 접촉비 개선
3. 치형 수정: 크라우닝, 릴리프
4. 정밀도 향상: JIS 2급 → 1급
5. 적절한 백래시: 0.05~0.2mm
```

**제조적 대책:**
```
1. 치형 연삭: 표면조도 향상
2. 러핑: 치면 다듬기
3. 밸런싱: 동적 평형
4. 조립 정밀도: 중심거리, 평행도
```

### 진동 특성
```
맞물림 주파수: fm = z × n/60 (Hz)
z: 치수, n: 회전수 (rpm)

회피 조건:
- 운전주파수 ≠ 고유진동수
- 맞물림 주파수 ≠ 고유진동수
- 적절한 감쇠 설계
```

## 🏭 기어 제조와 정밀도

### 기어 가공 방법
**창성 가공 (Generating):**
```
호빙 (Hobbing):
- 나선 절삭공구 사용
- 연속 가공 가능
- 생산성 우수
- 스퍼, 헬리컬기어

셰이핑 (Shaping):
- 랙 형태 공구
- 내접기어 가능
- 단속 가공
- 정밀도 우수
```

**성형 가공 (Forming):**
```
밀링: 성형 커터 사용
프레스: 대량 생산용
소결: 분말 성형
주조: 대형 기어
```

### 기어 정밀도 (JIS B 1702)
```
정밀도 등급:
0급: 최고 정밀도 (측정기기)
1급: 정밀 기계 (공작기계)
2급: 일반 정밀 기계
3급: 일반 기계
4급: 농기계, 건설기계

주요 검사 항목:
- 피치 오차: 인접 피치 편차
- 프로파일 오차: 치형 곡선 편차  
- 리드 오차: 나선 방향 편차
- 편심: 중심축 어긋남
```

## ⚙️🔗 기어트레인 설계

### 기어트레인 종류
**고정축 기어트레인:**
```
단순 기어트레인: 2개 기어
복합 기어트레인: 3개 이상
분기 기어트레인: 동력 분산

전체 기어비: i = (z2/z1) × (z4/z3) × ...
```

**유성 기어트레인:**
```
구성:
- 태양기어 (Sun Gear)
- 유성기어 (Planet Gear)  
- 내접기어 (Ring Gear)
- 캐리어 (Carrier)

특징:
- 큰 감속비
- 컴팩트
- 동력 분산
- 높은 효율
```

### 설계 고려사항
```
1. 기어비 배분: 각 단별 최적 배분
2. 회전 방향: 필요한 출력 방향
3. 중간축 설계: 처짐, 비틀림 제한
4. 베어링 배치: 하중 분산, 조립성
5. 윤활 시스템: 각 기어쌍 윤활 확보
6. 케이스 설계: 강성, 열방산, 소음
```

## 🤖 특수 기어 시스템

### 하모닉 드라이브 (Harmonic Drive)
```
구조:
- 웨이브 제너레이터 (Wave Generator)
- 플렉스 스플라인 (Flex Spline)  
- 서큘러 스플라인 (Circular Spline)

특징:
- 매우 큰 감속비: 50:1 ~ 320:1
- 백래시 거의 0
- 높은 위치 정밀도
- 로봇 관절용

감속비: i = (Zc - Zf)/Zf
Zc: 서큘러 스플라인 치수
Zf: 플렉스 스플라인 치수
```

### 사이클로이드 감속기
```
원리: 사이클로이드 곡선 이용
특징:
- 높은 감속비: 10:1 ~ 120:1
- 내충격성 우수
- 컴팩트
- 높은 토크 밀도

용도: 로봇, 공작기계, 컨베이어
```

### 로봇용 기어 요구사항
```
1. 높은 감속비: 모터 고속 → 관절 저속
2. 낮은 백래시: 위치 정밀도
3. 높은 강성: 동적 응답성
4. 컴팩트: 공간 제약
5. 경량: 관성 모멘트 최소화
6. 높은 효율: 전력 소모 최소화
```
