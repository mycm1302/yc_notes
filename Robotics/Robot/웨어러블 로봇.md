# 웨어러블 로봇

> 상위: [[분야별 로봇]]

인간의 신체에 착용하는 생체역학적 로봇 시스템으로, 인체의 운동 기능을 보조, 증강, 또는 복원하는 목적으로 설계된 장치입니다. 외골격 로봇(Exoskeleton) 또는 바이오메카트로닉 시스템이라고도 불립니다.

## 1. 웨어러블 로봇 개요

### 1.1 웨어러블 로봇과 외골격의 정의

**웨어러블 로봇 (Wearable Robot)**은 인간의 신체 형태와 기능 주변에 설계된 메카트로닉 시스템으로, 착용자와 외부적으로 결합되는 인체의 분절과 관절에 대응하는 구조를 가집니다.

**역사적 발전**
- **1960년대**: General Electric의 Hardiman 프로젝트 - 최초의 전신 외골격
- **1970년대**: 군사적 응용을 위한 파워 증강 시스템 연구
- **1990년대**: 의료 재활 분야 적용 시작
- **2000년대**: 바이오메카트로닉스 학문 체계화
- **2010년대**: 상용화 제품 출현 (ReWalk, HAL, Ekso)

### 1.2 바이오메카트로닉스와 생체영감의 역할

**바이오메카트로닉스 (Biomechatronics)**는 생물학적 시스템과 기계적 시스템의 융합을 다루는 학문 분야입니다.

**핵심 원칙**
```
Human System (H) + Robot System (R) → Integrated System (HR)
```

**설계 철학**
1. **투명성 (Transparency)**: 로봇이 인간의 자연스러운 움직임을 방해하지 않음
2. **보완성 (Complementarity)**: 인간의 부족한 부분을 로봇이 보완
3. **안전성 (Safety)**: 인간-로봇 접촉에서 안전 보장
4. **적응성 (Adaptability)**: 개인차와 상황 변화에 대응

### 1.3 로봇 외골격 관련 기술들

**핵심 기술 영역**
- **센서 기술**: EMG, IMU, 힘/토크 센서, 뇌파(EEG)
- **액추에이터 기술**: 전기모터, 공압/유압, 시리즈 탄성 액추에이터
- **제어 기술**: 임피던스 제어, 적응 제어, 생체신호 기반 제어
- **인터페이스 기술**: 기계적 부착, 신호 인터페이스, 안전 시스템
- **전력 시스템**: 배터리, 에너지 회수, 전력 관리### 1.4 웨어러블 외골격의 분류: 응용 분야

**신체 부위별 분류**
- **상지 외골격 (Upper-limb Exoskeleton)**: 어깨, 팔꿈치, 손목, 손가락
- **하지 외골격 (Lower-limb Exoskeleton)**: 고관절, 무릎, 발목  
- **척추 외골격 (Spinal Exoskeleton)**: 허리와 등 지지
- **전신 외골격 (Full-body Exoskeleton)**: 상하체 통합 시스템

**구동 방식별 분류**
- **능동형 (Active)**: 외부 동력으로 구동하는 시스템
- **수동형 (Passive)**: 스프링, 댐퍼 등 기계적 에너지 저장 요소 활용
- **준능동형 (Semi-active)**: 능동과 수동의 하이브리드 방식
- **소프트 (Soft)**: 유연한 재료와 공압/케이블 구동

**응용 분야별 분류**
- **의료용 (Medical)**: 재활, 보행 훈련, 치료 지원
- **산업용 (Industrial)**: 작업자 부담 경감, 생산성 향상
- **군사용 (Military)**: 전투원 능력 증강, 장비 운반
- **일상용 (Daily)**: 고령자 지원, 장애인 보조

## 2. 생물학적 영감과 생체모방 설계

### 2.1 생물학적 설계의 일반 원칙

**자연계의 설계 원칙**
생물학적 시스템은 수백만 년의 진화를 통해 최적화된 설계를 보여줍니다.

**효율성 원칙**
- **에너지 최소화**: 최소한의 에너지로 최대 효과
- **재료 최적화**: 무게 대비 강도 최대화
- **다기능성**: 하나의 구조로 여러 기능 수행

**적응성 원칙**  
- **환경 적응**: 변화하는 조건에 실시간 대응
- **학습 능력**: 경험을 통한 성능 개선
- **복구 능력**: 손상 시 자가 치유 메커니즘

### 2.2 생물학적 영감 설계의 개발

**바이오미메틱스 접근법**
1. **생물학적 시스템 연구**: 자연계 메커니즘 분석
2. **핵심 원리 추출**: 공학적 모델링 가능한 원리 도출
3. **공학적 구현**: 인공 시스템으로 실현
4. **성능 검증**: 생물학적 시스템과 비교 평가### 2.3 공학 설계에서 생물학적 영감의 수준

**Level 1: 형태적 모방 (Morphological Mimicry)**
- 생물학적 구조의 물리적 형태 복사
- 예: 새의 날개 형태를 모방한 비행체

**Level 2: 기능적 모방 (Functional Mimicry)**  
- 생물학적 기능과 메커니즘 구현
- 예: 근육의 수축-이완 메커니즘을 모방한 인공근육

**Level 3: 원리적 모방 (Principle-based Mimicry)**
- 생물학적 제어 원리와 정보처리 방식 적용
- 예: 신경계의 학습 알고리즘을 모방한 적응 제어

### 2.4 CPG와 반사를 이용한 생체영감 제어

**중추 패턴 생성기 (Central Pattern Generators, CPGs)**
CPG는 외부 감각 입력 없이도 리듬적 운동 패턴을 생성하는 신경 회로입니다.

**CPG의 특성**
- **자율적 리듬 생성**: 외부 입력 없이도 주기적 신호 생성
- **상호 억제**: 길항근 간의 교대 활성화 패턴
- **가소성**: 학습과 적응을 통한 패턴 수정

**CPG 수학적 모델**
```
Hopf 진동자 모델:
ẋ = (μ - x² - y²)x - ωy
ẏ = (μ - x² - y²)y + ωx
```

**반사 메커니즘 (Reflex Mechanisms)**
- **신장 반사**: 근육 길이 변화에 대한 즉각적 대응
- **힘 반사**: 과도한 힘에 대한 보호적 반응
- **균형 반사**: 자세 안정성 유지

**웨어러블 로봇에서의 적용**
```
보행 제어 = CPG_기본패턴 + 반사_조정 + 상위_명령
τ_total = τ_CPG + τ_reflex + τ_voluntary
```

### 2.5 내부 모델과 운동 제어

**순방향 모델 (Forward Model)**
```
예측_상태 = f(현재_상태, 운동_명령)
```

**역방향 모델 (Inverse Model)**  
```
운동_명령 = f⁻¹(현재_상태, 목표_상태)
```## 3. 웨어러블 로봇의 기구학과 동역학

### 3.1 로봇 역학: 운동 방정식

**라그랑지안 접근법**
```
L = T - V  (운동에너지 - 위치에너지)
d/dt(∂L/∂q̇) - ∂L/∂q = τ
```

**일반화된 운동 방정식**
```
M(q)q̈ + C(q,q̇)q̇ + g(q) = τ + J^T F_ext
```
여기서:
- M(q): 관성 행렬
- C(q,q̇): 코리올리/원심력 행렬  
- g(q): 중력 벡터
- τ: 관절 토크
- J^T F_ext: 외부 힘

### 3.2 인체 생체역학

**인체 관절 모델링**
각 관절에 대해 Denavit-Hartenberg 규약 적용:
```
T_i^{i-1} = Rot(z, θ_i) Trans(z, d_i) Trans(x, a_i) Rot(x, α_i)
```

**주요 인체 관절 특성**
- **어깨 관절**: 3자유도 구형 관절 (Spherical Joint)
- **팔꿈치 관절**: 1자유도 회전 관절 (Revolute Joint)
- **고관절**: 3자유도 구형 관절
- **무릎 관절**: 1자유도 회전 관절 (약간의 병진 포함)

**Hill 근육 모델**
```
F_muscle = F_0 · f_l(l) · f_v(v) · a(t)
```
- F_0: 최대 등척성 힘
- f_l(l): 길이-장력 관계
- f_v(v): 속도-장력 관계  
- a(t): 신경 활성화 수준### 3.3 외골격 시스템의 키네마틱 여용성

**여용성의 정의**
키네마틱 여용성은 로봇이 작업 공간에서 동일한 최종 효과기 위치를 달성하는 데 필요한 것보다 더 많은 자유도를 가질 때 발생합니다.

**여용성 해석**
```
q̇ = J⁺ẋ + (I - J⁺J)q̇_0
```
여기서:
- J⁺: 의사역행렬 (pseudoinverse)
- (I - J⁺J): 영공간 투영 행렬
- q̇_0: 임의의 관절 속도

**외골격에서의 여용성 활용**
1. **인체 편안함 최적화**: 관절 부담 최소화
2. **특이점 회피**: 특이 구성 회피
3. **보조 목표 달성**: 에너지 최소화, 조작성 최대화

**최적화 기반 여용성 해결**
```
minimize: ||W(q - q_ref)||²
subject to: J(q)q̇ = ẋ_desired
```

### 3.4 인체 사지와 웨어러블 로봇 간의 키네마틱 호환성

**키네마틱 비호환성의 원인**
1. **관절축 정렬 불일치**: 인체와 로봇 관절의 회전축 차이
2. **관절 범위 차이**: 인체와 로봇의 가동 범위 불일치  
3. **분절 길이 차이**: 개인차에 따른 사지 비율 차이

**비호환성의 부정적 효과**
- **접촉 압력 증가**: 피부 손상과 불편함
- **운동 제한**: 자연스러운 움직임 방해
- **안전성 저하**: 예상치 못한 힘과 토크 발생

**키네마틱 호환성 극복 방법**
1. **자기 정렬 관절 (Self-aligning joints)**
2. **순응성 인터페이스 (Compliant interfaces)**  
3. **가변 구조 설계 (Variable structure design)**
4. **개인 맞춤 캘리브레이션**## 4. 인체-로봇 인지적 상호작용 (cHRI)

### 4.1 인체-로봇 상호작용 개요

**인지적 상호작용 (Cognitive HRI)**은 인간의 의도, 감정, 인지 상태를 파악하여 로봇이 적절히 반응하는 것을 의미합니다.

**상호작용 채널**
1. **생체전기 신호**: EEG, EMG, EOG
2. **생체역학 신호**: 힘, 위치, 속도, 가속도
3. **생리학적 신호**: 심박수, 호흡, 피부 전도도

### 4.2 뇌 활동의 생체전기 모니터링을 통한 cHRI

**뇌 활동의 생리학**
- **뉴런의 활동전위**: 신경 신호의 기본 단위
- **피질 리듬**: Alpha(8-12Hz), Beta(13-30Hz), Gamma(>30Hz)
- **사건 관련 전위 (ERP)**: 특정 자극에 대한 뇌 반응

**뇌전도 (EEG) 모델과 매개변수**
```
EEG 신호 모델:
x(t) = Σ s_i(t) · a_i + n(t)
```
여기서:
- s_i(t): i번째 뇌 소스 신호
- a_i: 소스에서 전극까지의 가중치
- n(t): 노이즈

**뇌 제어 인터페이스: 접근법과 알고리즘**
1. **운동 상상 (Motor Imagery)**: 실제 움직임 없이 운동 생각
2. **P300 전위**: 희귀 자극에 대한 반응
3. **SSVEP**: 깜빡이는 자극에 대한 정상상태 반응

**특징 추출 알고리즘**
- **공간 필터링**: CSP (Common Spatial Patterns)
- **주파수 분석**: FFT, 웨이블릿 변환
- **분류**: SVM, LDA, 신경망

### 4.3 근육 활동의 생체전기 모니터링을 통한 cHRI

**근육 활동의 생리학**
- **운동 단위**: 하나의 운동 뉴런과 지배받는 근섬유들
- **모집 원리**: 작은 운동 단위부터 순차적 활성화
- **발화 빈도**: 힘 조절의 주요 메커니즘**근전도 (EMG) 모델과 매개변수**
```
EMG 신호 처리 과정:
Raw EMG → BPF[20-500Hz] → Rectification → LPF[5Hz] → Normalization
```

**표면 EMG 신호 특징 추출**
1. **시간 영역 특징**
   - **MAV**: ```MAV = (1/N) Σ|x_i|```  
   - **RMS**: ```RMS = √((1/N) Σx_i²)```
   - **영점 교차 (ZC)**: 신호 복잡도 지표

2. **주파수 영역 특징**
   - **중간 주파수 (MDF)**: 파워 스펙트럼의 중간값
   - **평균 주파수 (MNF)**: 파워 가중 평균 주파수

**EMG 활동 분류**
```
패턴 분류 과정:
특징벡터 → 정규화 → 분류기 → 운동 의도
```

**힘과 토크 추정**
```
근육-관절 토크 관계:
τ = R(q) · F_muscle
여기서 R(q)는 모멘트 암 행렬
```

### 4.4 생체역학적 모니터링을 통한 cHRI

**생체역학적 모델과 매개변수**
1. **관절 각도와 각속도**: 엔코더, IMU 센서
2. **접촉력과 압력**: 로드셀, 압력 센서
3. **근육 활성화 패턴**: EMG 기반 추정

**생체역학 제어 인터페이스**
```
의도 추정 = f(관절각도, 관절토크, EMG패턴, 접촉력)
```

**접근법과 알고리즘**
- **임피던스 추정**: 인체의 기계적 특성 실시간 추정
- **의도 분류**: 기계학습 기반 운동 의도 파악
- **적응적 보조**: 사용자 상태에 따른 보조 수준 조절## 5. 인체-로봇 물리적 상호작용 (pHRI)

### 5.1 물리적 상호작용 개요

**생리학적 요인**
- **피부 탄성**: 압력 분산과 접촉 편안함
- **근골격계 강성**: 관절별 가변 임피던스
- **감각 피드백**: 촉각, 고유수용감각

**웨어러블 로봇 설계 측면**
- **인터페이스 설계**: 접촉 면적과 압력 분산
- **재료 선택**: 생체 적합성과 내구성
- **기계적 호환성**: 인체와 로봇의 임피던스 매칭

### 5.2 인체 하중 적용

**하중 적용 방식**
1. **점 접촉 (Point Contact)**: 집중된 힘 전달
2. **면 접촉 (Surface Contact)**: 분산된 압력 적용
3. **연속 접촉 (Continuous Contact)**: 움직임 전반에 걸친 접촉

**압력 분산 원리**
```
P = F/A
압력 최소화 = 접촉 면적 최대화 + 힘 분산
```

**생체역학적 제약**
```
압력 제한: P_contact ≤ P_max (통증 임계값)
힘 제한: |F_interaction| ≤ F_safe
시간 제한: 장시간 접촉 시 압력 감소 필요
```

### 5.3 인체-로봇 상호작용 제어

**결합된 동역학 시스템**
```
[M_h + M_r](q)q̈ + [C_h + C_r](q,q̇)q̇ + [g_h + g_r](q) = τ_h + τ_r + τ_int
```

**임피던스 제어**
로봇이 원하는 기계적 임피던스를 보이도록 제어:
```
F_int = M_d(ẍ_d - ẍ) + B_d(ẋ_d - ẋ) + K_d(x_d - x)
```

**어드미턴스 제어**
상호작용 힘을 입력으로 받아 로봇의 움직임을 생성:
```
ẍ = M_d^(-1)[F_int - B_d ẋ - K_d x]
```## 6. 웨어러블 로봇 기술들

### 6.1 센서 기술

**관성 측정 장치 (IMU)**
```
센서 융합: 자이로스코프 + 가속도계 + 지자기계 → 자세 추정
상보 필터: θ = α(θ + ω·dt) + (1-α)θ_acc
```

**힘/토크 센서**
- **6축 로드셀**: F_x, F_y, F_z, M_x, M_y, M_z 동시 측정
- **캘리브레이션**: ```F_actual = C · F_raw + offset```

**생체신호 센서**
- **EMG 센서**: 근육 활성화 패턴 측정
- **EEG 센서**: 뇌파 신호 획득
- **압력 센서**: 접촉 압력 분포 측정

### 6.2 액추에이터 기술

**전기 모터 기반 시스템**
```
DC 모터 동역학:
전기: V = L(di/dt) + Ri + K_e ω
기계: J ω̇ = K_t i - τ_load - B ω
```

**시리즈 탄성 액추에이터 (SEA)**
```
τ_output = K_s (θ_motor/N - θ_output)
장점: 충격 흡수, 힘 제어 정확도, 안전성
```

**공압 인공근육 (PAM)**
```
F = P · A_eff - F_elastic
수축률 = (L_0 - L) / L_0
```### 6.3 휴대용 에너지 저장 기술

**배터리 시스템**
- **리튬이온**: 높은 에너지 밀도, 경량화
- **리튬폴리머**: 유연한 형태, 안전성 개선
- **연료전지**: 장시간 작동, 빠른 충전

**에너지 회수 시스템**
```
회생 제동: E_recovered = ∫ τ_brake · ω dt
탄성 에너지 저장: E_spring = ½kx²
```

**전력 관리**
- **효율적 모터 드라이버**: PWM 제어, 낮은 손실
- **절전 모드**: 비활성 시 전력 소모 최소화
- **적응적 보조**: 필요시에만 전력 사용

### 7.1 유선에서 무선 웨어러블 로봇 네트워크로

- **Bluetooth**: 저전력, 단거리 통신
- **WiFi**: 고속 데이터, 인터넷 연결
- **Zigbee**: 메시 네트워크, 저전력
- **LoRa**: 장거리, 저전력 광역 네트워크

- **전도성 섬유**: 신호 전송과 전력 공급
- **압전 섬유**: 변형 감지, 에너지 하베스팅

### 9.1 계층적 제어 구조

**고수준 제어 (High-Level Control)**
1. **작업 계획**: 사용자 의도 파악, 동작 모드 결정
2. **의도 인식**: ```Intent = Classifier(EMG, IMU, Force, Context)```

**중수준 제어 (Mid-Level Control)**  
1. **궤적 생성**: ```q_d(t) = a_0 + a_1t + ... + a_nt^n```
2. **보조 전략**: 투명/보조/저항 모드 선택

**저수준 제어 (Low-Level Control)**
```
토크 제어: τ = τ_ff + τ_fb
τ_ff = M(q)q̈_d + C(q,q̇)q̇_d + g(q)
τ_fb = K_p(q_d - q) + K_d(q̇_d - q̇)
```

### 9.2 적응형 제어 기법

**모델 기준 적응 제어 (MRAC)**
```
θ̇ = -Γ φ(x) e^T P B
```

**슬라이딩 모드 제어 (SMC)**
```
s = ė + λe  (슬라이딩 면)
τ = τ_eq + τ_dis  (제어 입력)
```

### 9.3 안전 시스템 설계

**본질적 안전 (Intrinsic Safety)**
- 기계적 스토퍼로 관절 범위 제한
- 탄성 요소를 통한 충격 완화
- 저관성 설계로 충돌 에너지 최소화

**안전 제약 조건**
```
기하학적: q_min ≤ q ≤ q_max
동력학적: |τ| ≤ τ_max, |F_int| ≤ F_max
```## 10. 성능 평가 및 미래 동향

### 10.1 정량적 평가 지표

**투명성 (Transparency)**
```
Transparency = 1 - |τ_interaction|/|τ_human| × 100%
```

**보조 효과 (Assistance Effectiveness)**
```
Assistance = (EMG_baseline - EMG_assisted)/EMG_baseline × 100%
```

**에너지 효율성**
```
Efficiency = W_useful / W_total × 100%
```

### 10.2 임상 평가

**기능 평가**
- **FIM**: Functional Independence Measure
- **보행 분석**: 속도, 보폭, 대칭성
- **근력 측정**: 등척성/등속성 근력

### 10.3 미래 기술 동향

**뇌-컴퓨터 인터페이스 (BCI)**
- EEG/fNIRS 기반 의도 파악
- 운동 상상 신호 활용
- 실시간 뇌파 디코딩

**인공지능 융합**
- 강화학습 기반 적응 제어
- 개인별 맞춤 학습
- 예측적 보조 제어

**상용화 과제**
- **경량화**: 현재 5-15kg → 목표 2kg 이하
- **배터리**: 현재 4-8시간 → 목표 24시간 이상  
- **비용**: 현재 $10,000-60,000 → 목표 $5,000 이하
