## 🎯 기본 제어 구조

### 피드백 시스템
- **목표값** → **제어기** → **시스템** → **출력**
- **피드백**: 출력을 측정하여 입력에 반영
- **오차**: 목표값과 실제값의 차이

### 제어 성능
- **안정성**: 시스템이 발산하지 않음
- **정확성**: 정상상태 오차 최소화
- **응답성**: 빠른 과도응답
