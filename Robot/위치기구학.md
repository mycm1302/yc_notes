# 위치기구학

> 상위: [[기구학]]

로봇의 위치와 자세를 다루는 기본적인 기구학 분야입니다.

## 📚 하위 영역

### [[회전표현]]
3차원 공간에서 로봇의 자세를 표현하는 다양한 방법들

### [[변환표현]]
위치와 자세를 통합적으로 나타내는 변환 행렬

### [[순기구학]]
관절 변수에서 말단 위치/자세를 구하는 순방향 문제

### [[역기구학]]
말단 위치/자세에서 관절 변수를 구하는 역방향 문제

---

## 🎯 핵심 흐름

### 표현 → 순기구학 → 역기구학
1. 회전/변환 표현 방법 학습
2. 순기구학으로 기본 관계 이해
3. 역기구학으로 실제 문제 해결

### 수학적 도구
- 회전행렬 (SO(3))
- 동차변환 (SE(3))
- 스크류 이론
- 수치해법
