# 에너지방법

> 상위: [[동역학]]

에너지 보존과 에너지 기반 해석 방법들입니다.

## ⚡ 에너지 유형

### 기본 에너지
- [[에너지]] - 에너지의 기본 개념과 보존 법칙
- [[운동에너지]] - 로봇의 운동에 의한 에너지
- [[위치에너지]] - 중력장과 탄성 에너지

### 힘의 분류
- [[보존력]] - 위치에만 의존하는 힘 (중력, 탄성력)
- [[소산력]] - 에너지를 소모하는 힘 (마찰, 댐핑)

## 🔗 수학적 관계

### 에너지 함수
- **라그랑지안**: L = T - V
- **해밀토니안**: H = T + V
- **총 에너지**: E = T + V

### 보존 법칙
- 보존계: dE/dt = 0
- 소산계: dE/dt ≤ 0
- 입력 시스템: dE/dt = P_input
